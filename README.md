lipkart Sales Performance & Customer Insights Analysis

Project Overview This project provides a comprehensive exploratory data analysis (EDA) of sales data from Flipkart, one of India's leading e-commerce platforms. Using a dataset of 1,000 transactions, the analysis explores key performance indicators (KPIs) such as total sales, profit margins, and customer satisfaction across various categories and regions.

**Key Objectives**

Sales Trend Analysis: Identifying seasonal patterns and high-performing months throughout the year.

Customer Segmentation: Analyzing purchasing behavior across Online, Wholesale, and Retail segments.

Category Performance: Evaluating which product categories (Books, Clothing, Electronics, etc.) drive the most revenue and profit.

Geographic Insights: Mapping sales performance across different Indian regions (North, South, East, West).

Discount & Rating Correlation: Investigating how discount percentages impact customer ratings and overall sales volume.

Technical Stack

Language: Python

**Libraries**:

Pandas: For data cleaning and manipulation.

NumPy: For numerical computations.

Matplotlib & Seaborn: For creating insightful visualizations and statistical charts.
